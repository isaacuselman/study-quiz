# Study Quiz

A self-study quiz app with spaced repetition for CS50 and CS50P courses. Quiz yourself on lecture material, track your progress, and use SM-2 spaced repetition to reinforce what you missed.

## Disclaimers

- **This is NOT an official Harvard or CS50 product.** It is an independent study tool with no affiliation to Harvard University, CS50, or David Malan.
- **Questions are AI-generated** from lecture transcripts and may contain errors, inaccuracies, or outdated information. Always refer to the official course materials as the authoritative source.
- **For personal study use only.** This tool is meant to supplement your learning, not replace watching the lectures or doing the problem sets.

## Features

- 700+ multiple-choice questions across CS50 and CS50P
- Organized by course and lecture week
- SM-2 spaced repetition for missed questions
- Progress tracking (mastered / in review / unseen) per lecture
- Study streak counter
- Keyboard shortcuts (1-4 to select, Space/Enter to submit)
- Works offline (all data stored in browser localStorage)
- Dark theme

## Getting Started

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal (default: http://localhost:5173).

## Adding Your Own Questions

1. Get a transcript of the lecture you want to study
2. Use the prompt in `QUIZ_PROMPT_TEMPLATE.md` with any LLM (Claude, ChatGPT, etc.)
3. Copy the generated JSON object
4. Paste it into the appropriate course file in `src/data/`
5. Run `npm run dev` to verify it shows up

## Tech Stack

- React 19 + Vite
- No backend — runs entirely in the browser
- SM-2 algorithm for spaced repetition
- Fisher-Yates shuffle for randomization
